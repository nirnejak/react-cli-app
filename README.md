# React CLI App

A CLI tool built using React(Ink)

## Authors

- [@nirnejak](https://www.github.com/nirnejak)

## Run Locally

Clone the project

```bash
  git clone https://github.com/nirnejak/react-cli-app
```

Go to the project directory

```bash
  cd react-cli-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
